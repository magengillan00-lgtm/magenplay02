# 🚀 كيفية بناء Magen Play 02 وإنشاء APK

## الحل السهل: GitHub Actions (مجاناً)

المشروع مُعدّ للبناء التلقائي! بمجرد رفعه إلى GitHub:

1. **البناء التلقائي** - سيبدأ عند رفع الكود
2. **الحصول على APK** - في تبويب Actions أو Releases

## الطريقة اليدوية: Android Studio

### الخطوات:
1. استنسخ المشروع:
```bash
git clone https://github.com/magengillan00/magenplay02.git
cd magenplay02
```

2. افتح Android Studio
3. اختر **Open Project** وحدد مجلد المشروع
4. انتظر حتى ينتهي Gradle من تحميل التبعيات
5. اضغط **Build > Build Bundle(s) / APK(s) > Build APK(s)**

APK سيكون في: `app/build/outputs/apk/release/app-release.apk`
